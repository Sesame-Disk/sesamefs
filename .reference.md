## Testing Sync Protocol

```bash
# Get sync token for a library
export REPO_ID="aa89cce8-f2f1-44ad-98f0-1ea87ca6a3ed"
export SYNC_TOKEN=$(curl -s -H "Authorization: Token $TOKEN" \
  "https://app.nihaoconsult.com/api2/repos/$REPO_ID/download-info/" | \
  python3 -c "import sys,json; print(json.load(sys.stdin)['token'])")

echo "Sync Token: $SYNC_TOKEN"

# Get HEAD commit
curl -s -H "Seafile-Repo-Token: $SYNC_TOKEN" \
  "https://app.nihaoconsult.com/seafhttp/repo/$REPO_ID/commit/HEAD"

# Get specific commit (includes root_id)
COMMIT_ID="3a7d261b4f81a77b1f0594a3435975c77d84f78f"
curl -s -H "Seafile-Repo-Token: $SYNC_TOKEN" \
  "https://app.nihaoconsult.com/seafhttp/repo/$REPO_ID/commit/$COMMIT_ID"

# Get FS ID list
curl -s -H "Seafile-Repo-Token: $SYNC_TOKEN" \
  "https://app.nihaoconsult.com/seafhttp/repo/$REPO_ID/fs-id-list/?server-head=$COMMIT_ID"

# Check FS objects exist
curl -s -X POST -H "Seafile-Repo-Token: $SYNC_TOKEN" \
  "https://app.nihaoconsult.com/seafhttp/repo/$REPO_ID/check-fs" \
  -d '["aceaa31c23a2b57cb577834ebd6f52cb2fbddec9"]'
```

## Testing Binary Formats

```bash
# Test pack-fs and examine binary response
ROOT_ID="aceaa31c23a2b57cb577834ebd6f52cb2fbddec9"
curl -s -X POST -H "Seafile-Repo-Token: $SYNC_TOKEN" \
  -H "Content-Type: application/json" \
  "https://app.nihaoconsult.com/seafhttp/repo/$REPO_ID/pack-fs/" \
  -d "[\"$ROOT_ID\"]" | xxd | head -30

# Decompress pack-fs response (skip 44-byte header: 40 ID + 4 size)
curl -s -X POST -H "Seafile-Repo-Token: $SYNC_TOKEN" \
  "http://localhost:8080/seafhttp/repo/$REPO_ID/pack-fs" \
  -d "[\"$ROOT_ID\"]" | tail -c +45 | python3 -c "import sys,zlib; print(zlib.decompress(sys.stdin.buffer.read()).decode())"
```

## Key API Format Notes

| Endpoint | Field | Seafile Format |
|----------|-------|----------------|
| `/commit/{id}` | `parent_id` | `null` (not `""`) |
| `/commit/{id}` | `version` | `1` (not `0`) |
| `/commit/{id}` | `creator` | 40 zeros |
| `/fs-id-list` | Response | `[]` (JSON array) |
| `/permission-check` | Response | Empty body |
| `/pack-fs` | Request | JSON array: `["id1", "id2"]` |
| `/pack-fs` | Response | Binary: `[40-byte ID][4-byte size BE][zlib JSON]...` |
| `/check-fs` | Request | JSON array: `["id1", "id2"]` |
| `/check-fs` | Response | JSON array of missing IDs |
